# metallb

MetalLB provider role for DMF ingress modes.

Supports:

- `metallb-l2`
- `metallb-bgp`

The generic `ingress` role selects this provider when the environment chooses a
MetalLB-backed north-south path.

**Status:** STUB — not yet implemented.
